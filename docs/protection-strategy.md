# Project Protection Strategy

Status: **1.0.0 protection strategy**

This document summarizes project protection and distribution guidance for City Life Core.

## Guidance

- Publish official releases from the canonical repository.
- Keep checksums with release archives where possible.
- Prefer source builds or verified SDK ZIP packages.
- Document accepted validation gaps in release manifests.
- Keep license and provenance information visible in distributed archives.
