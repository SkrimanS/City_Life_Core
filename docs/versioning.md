# Versioning Policy

Status: **1.0.0 versioning policy**

City Life Core uses semantic versioning for released SDK lines.

## Rules

- Patch releases fix bugs or documentation issues without intentional public API breaks.
- Minor releases may add APIs and workflows while preserving the supported 1.x source contract.
- Major releases may introduce migration-required changes.
- Release notes must call out API, persistence, package-layout and validation changes.
